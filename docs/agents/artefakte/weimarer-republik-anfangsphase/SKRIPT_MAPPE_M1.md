# SKRIPT_MAPPE_M1 — November 1918: Eine neue Ordnung

**Aktenkarton-Titel:** "Akte 1918 — Wie wurde aus dem Kaiserreich eine Republik?"

---

## 1. Einstiegs-Szene (Erzähl-Hook)

Ein verstaubter Aktenkarton steht auf dem Redaktionstisch. Auf dem Deckel steht: "Berlin, November 1918". Darin: ein Foto, eine Karte, eine handschriftliche Notiz und ein Aufruf. Das Foto zeigt einen Mann am Fenster eines großen Gebäudes. Eine Menschenmenge schaut hinauf. Er ruft etwas. Was?

**Einstiegs-Frage (sichtbar):**
> "An einem einzigen Tag wurde Deutschland zweimal zur Republik ausgerufen. Wie konnte das passieren — und was bedeutete es?"

**Hook-Funktion:** Neugier wecken durch ein Paradox. Ein Land, eine Republik — aber zwei Ausrufungen. Wieso?

## 4. Mappenabschluss-Beschreibung (Sicherungsfunktion)

**Ergebnis-Notiz für die Sonderausgabe (Tafelbild-Saat für M1):**

| Datum | 9. November 1918 |
|---|---|
| Ort | Berlin |
| Akteure | Scheidemann (MSPD) am Reichstag; Liebknecht (Spartakusbund) am Stadtschloss |
| Was geschah | Zwei Ausrufungen einer Republik — eine parlamentarisch, eine sozialistisch |
| Was änderte sich | Kaiser dankte ab. Wahlrecht für Frauen angekündigt. Acht-Stunden-Tag versprochen. |

**Sicherungs-Erkenntnis:**
Am 9. November 1918 endete die Monarchie. Eine Republik begann. Es gab zwei Vorstellungen davon, wie sie aussehen sollte. Beide Seiten waren sich einig: Das alte System war vorbei.

**Übergang zu M2 (Sandwich):**
Die Republik war ausgerufen. Aber wie sollte sie funktionieren? Eine neue Ordnung braucht Regeln. Die Antwort kam wenige Monate später aus Weimar.
