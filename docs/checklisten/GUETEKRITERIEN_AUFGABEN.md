## 1. Leitsatz

> "Anforderungsbereich I: reine Wiedergabe in bekannten Zusammenhaengen. Anforderungsbereich II: selbststaendiges Bearbeiten bekannter Inhalte. Anforderungsbereich III: reflexiver Umgang mit neuen Problemstellungen."
> — LLZ Uni Halle, AFB-Taxonomie

> "Nicht was getan wird, sondern was gelernt wird."
> — Kernprinzip der Lernzielformulierung (Rechercheergebnisse Lernziele)

Aufgaben in Escape-Games sind keine Wissensabfragen. Sie sind **diagnostische Instrumente**, die pruefen, ob die durch Materialarbeit angestrebten Lernzuwuechse tatsaechlich eingetreten sind. Eine gute Aufgabe ist praezise auf ein Lernziel kalibriert, kognitiv aktivierend und aus dem bereitgestellten Material beantwortbar. Die Schwierigkeitsprogression innerhalb einer Mappe bildet den Lernweg vom Erinnern (AFB I) ueber Verstehen und Anwenden (AFB II) bis zum Beurteilen (AFB III) ab.

## 2. Fachdidaktische Fundierung

### 2.1 Anforderungsbereiche als Aufgaben-Taxonomie

Die drei AFB definieren, welche kognitive Leistung eine Aufgabe verlangt. Jeder AFB hat spezifische Operatoren und geeignete Aufgabenformate:

| AFB | Kognitive Leistung | Operatoren (GPG) | Geeignete Aufgabentypen |
|---|---|---|---|
| **I** (Reproduktion) | Wiedergabe in bekanntem Zusammenhang | benennen, beschreiben, nennen, lokalisieren, darstellen | multiple-choice (Faktenwissen), zuordnung (Kategorien), lueckentext (Fachbegriffe) |
| **II** (Reorganisation/Transfer) | Selbststaendiges Bearbeiten, Uebertragen | analysieren, einordnen, erklaeren, erlaeutern, vergleichen, gegenueberstellen | reihenfolge (Chronologie/Prozesse), zuordnung (Zusammenhaenge), lueckentext (Zusammenhaenge) |
| **III** (Reflexion/Problemloesung) | Reflexiver Umgang, eigene Stellungnahme | beurteilen, bewerten, diskutieren, eroertern, Stellung nehmen, reflektieren, entwickeln | freitext-code (Stellungnahme, Problemloesung) |

**Konsequenz:** Der AFB einer Aufgabe wird nicht durch den Aufgabentyp allein bestimmt, sondern durch das Zusammenspiel von Fragestellung + Aufgabentyp + geforderter kognitiver Leistung. Eine MC-Frage kann AFB II erreichen, wenn sie Transferleistung verlangt.

### 2.2 Fragetypen nach kognitiver Funktion

Ulrich (2016) unterscheidet Fragenformen nach ihrer kognitiven Funktion. Fuer Escape-Game-Aufgaben relevant:

| Fragetyp                         | Funktion                                               | Eignung fuer Escape-Games              |
| -------------------------------- | ------------------------------------------------------ | -------------------------------------- |
| **Geschlossene Alternativfrage** | Entscheidung zwischen vorgegebenen Optionen            | MC, Zuordnung — Basis jeder Mappe      |
| **Offene/oeffnende Frage**       | Einfuehrung oder Reflexion, erfordert komplexe Antwort | Freitext — Abschluss der Mappe         |
| **Anwendungsfrage**              | Wissen in neuem Kontext anwenden                       | AFB II — Transfer innerhalb der Mappe  |
| **Vergleichsfrage**              | Wissensteile miteinander verknuepfen                   | AFB II — Zuordnung, Reihenfolge        |
| **Schlussfolgerungsfrage**       | Vom Bekannten auf Neues schliessen                     | AFB II-III — Freitext, MC mit Transfer |
| **Problemloesefrage**            | Gelerntes auf Problem anwenden                         | AFB III — Freitext                     |
| Rhetorische Frage                | Kein Denken angeregt                                   | **NICHT VERWENDEN**                    |
| Suggestivfrage                   | Antwort vorweggenommen                                 | **NICHT VERWENDEN**                    |

### 2.3 Operatoren-Praezision

Die Rechercheergebnisse identifizieren Anti-Patterns bei Aufgabenformulierungen, die direkt auf Escape-Game-Fragen uebertragbar sind:

| Anti-Pattern | Beispiel | Korrektur |
|---|---|---|
| Nicht-operationalisierbare Verben | "Was weisst du ueber den Imperialismus?" | "**Nenne** zwei Ursachen des Imperialismus." |
| Doppelte kognitive Anforderung | "Beschreibe und beurteile die Buendnispolitik." | Zwei separate Aufgaben: Beschreiben (AFB I) + Beurteilen (AFB III) |
| Zu global fuer eine Aufgabe | "Erklaere den Ersten Weltkrieg." | "Erklaere, warum die Buendnissysteme einen lokalen Konflikt zum Weltkrieg machten." |
| Aktivitaet statt Lernnachweis | "Lies den Quellentext und beantworte die Frage." | Direkte Frage ohne Anweisung zur Aktivitaet |

### 2.4 Aufgabe-Material-Kongruenz

Die **Material-Alignment-Pflicht** (AGENT_RAETSEL Aufgabe 1) verlangt, dass jede Aufgabe aus den bereitgestellten Materialien beantwortbar ist. Die fachdidaktische Fundierung praezisiert das:

- **Fachbegriff-Abdeckung:** Kein Fachbegriff in einer Aufgabe, der nicht in einem Material der Mappe eingefuehrt und erklaert wurde
- **Informations-Abdeckung:** Die zur Beantwortung noetige Information muss in den referenzierten Materialien enthalten sein — nicht nur thematisch, sondern konkret
- **Niveau-Kongruenz:** Die geforderte kognitive Leistung (AFB) muss durch die Materialarbeit vorbereitet sein. Eine AFB-III-Frage braucht Material, das Perspektiven/Kontroversen liefert

### 2.5 SCPL-Fragebogen-Modell

Der Fragebogen einer Mappe ist kein isoliertes Quiz, sondern ein **diagnostischer Spiegel des SCPL-Erarbeitungswegs**. Die SCPL-Struktur des Hefteintrags (Situation → Complication → Problem → Loesung) definiert den Lernweg; die Aufgabensequenz prueft, ob jede Station dieses Wegs verstanden wurde.

**Zonen-Zuordnung:**

| SCPL-Zone | Diagnostische Funktion | AFB-Bereich | Prueffokus |
|---|---|---|---|
| S (Situation) | Vorwissen aktivieren, Kontext sichern | AFB I | Fachbegriffe, Einordnung |
| C (Complication) | Pro Erarbeitungsschritt 1+ Aufgabe | AFB I-II (steigend) | Schritt-spezifisches Verstaendnis |
| P (Problem) | Problemverstaendnis pruefen | AFB II | Zusammenhaenge, Widersprueche |
| L (Loesung) | Synthese, Transfer, Beurteilung | AFB II-III | Gesamterkenntnis, Stellungnahme |

Die Aufgabenzahl (5-8) ergibt sich aus der Anzahl der C-Schritte plus je 1 Aufgabe fuer S, P, L. Komplexe C-Schritte koennen 2 Aufgaben erhalten; bei wenigen C-Schritten koennen diese zusammengefasst werden.

**Tafelbild-Bezug:** Das Tafelbild ist die Quintessenz des Lernzuwachses (vgl. G1). Mindestens 1 Aufgabe pro Mappe muss direkt auf einen TB-Knoten zielen. Die Freitext-Aufgabe (L-Zone) reflektiert die Gesamterkenntnis des Tafelbilds.

### 2.6 Besinnungsphasen und Aufgabendesign

GPG B1 unterscheidet sachbezogene Besinnung (Zusammenhaenge erkennen, Begriffe klaeren, Ursachen verstehen) und wertbezogene Besinnung (ethische Stellungnahme). Das SCPL-Modell bildet diese Progression ab:

- **S/C-Zonen** (AFB I-II): Sachbezogene Besinnung — Fakten sichern, Erarbeitungsschritte pruefen, Zusammenhaenge herstellen
- **P-Zone** (AFB II): Uebergang — Problemverstaendnis, Analyse, Anwendung
- **L-Zone** (AFB II-III): Wertbezogene Besinnung — Synthese, Transfer auf Gegenwart, Stellungnahme, Beurteilung

## 6. Operationalisierung

### 6.1 A1 AFB-Kongruenz — Operator-AFB-Abgleich

**Methode:** Pro Aufgabe den verwendeten Operator identifizieren und gegen die AFB-Operatoren-Tabelle (Abschnitt 2.1) pruefen. Dann die tatsaechliche kognitive Anforderung der Aufgabe klassifizieren: Verlangt sie Wiedergabe (AFB I), Transfer (AFB II) oder Reflexion (AFB III)?

**FAIL-Muster:** Aufgabe deklariert AFB II, fragt aber: "Welche der folgenden Aussagen ist richtig?" mit 4 Optionen, von denen nur eine woertlich im Material steht → tatsaechlich AFB I (Wiedergabe).
**PASS-Muster:** Aufgabe deklariert AFB II, fragt: "Welche der folgenden Aussagen erklaert am besten, warum..." → erfordert Verstaendnis eines Zusammenhangs, tatsaechlich AFB II.

### 6.2 A2 Fragestaemme-Klarheit — Checkliste pro Aufgabe

**Methode:** Pro Aufgabe pruefen:
- [ ] Genau EINE kognitive Anforderung? (Kein "und" zwischen zwei Operatoren)
- [ ] Keine Negation ohne Hervorhebung? ("Welche Aussage ist **NICHT** korrekt?")
- [ ] Keine Doppeldeutigkeit? (Nur eine sinnvolle Interpretation moeglich)
- [ ] Keine ueberflüssigen Informationen im Fragestamm?
- [ ] Frage verstaendlich ohne Fachvorwissen, das ueber das Material hinausgeht?
- [ ] **Inhaltliche Verankerung (v3.4):** Mindestens 1 konkretes Element (Person, Ort, Gegenstand, Ereignis)? Keine reinen Metabegriffe ohne Bezug?

### 6.3 A4 Distractor-Qualitaet — MC-Pruefung

**Methode:** Pro MC-Aufgabe jeden Distraktor klassifizieren:

| Distractor-Typ | Qualitaet | Beispiel |
|---|---|---|
| Haeufige Fehlvorstellung | **GUT** | "Die Dampfmaschine" (wenn Antwort "Webstuhl" ist — beide sind Industrialisierungs-Erfindungen) |
| Teilwahrheit | **GUT** | "Frankreich und England" (wenn Antwort "Frankreich und Russland" ist — ein Partner stimmt) |
| Thematisch verwandt | **AKZEPTABEL** | Anderer historischer Begriff aus demselben Kontext |
| Offensichtlich absurd | **FAIL** | Anachronismus, sachfremder Begriff, humoristische Option |
| "Alle/Keine der obigen" | **FAIL** | Testet Ratestrategien, nicht Wissen |

**Schwellenwert:** Mindestens 2 von 3 Distraktoren muessen GUT oder AKZEPTABEL sein.

### 6.4 A5 Schwierigkeits-Progression — Monotonie-Pruefung

**Methode:** Alle Aufgaben einer Mappe (5-8) in eine AFB-Sequenz abbilden und auf Monotonie pruefen. Erste Aufgabe muss AFB I sein, letzte Aufgabe muss AFB III (Freitext) sein.

**PASS (5 Aufgaben):** I → I → II → II → III
**PASS (7 Aufgaben):** I → I → I-II → II → II → II → III
**PASS (6 Aufgaben):** I → I-II → II → II → II-III → III
**FAIL:** I → II → I → II → III (Regression bei Aufgabe 3)
**FAIL:** II → II → II → II → III (kein Einstieg auf AFB I)
**FAIL:** I → I → II → II → II (kein AFB-III-Abschluss)

### 6.5 A7 Operator-Praezision — Verb-Pruefung

**Methode:** Pro Aufgabe das Hauptverb der Fragestellung identifizieren. Steht es in der Operatoren-Tabelle (Abschnitt 2.1)? Wenn nicht: Ist es operationalisierbar (beobachtbares Verhalten)?

**FAIL-Verben:** "Was weisst du...", "Kennst du...", "Hast du verstanden...", "Denke an..."
**PASS-Verben:** "Nenne...", "Ordne zu...", "Erklaere...", "Vergleiche...", "Beurteile..."
**Grenzfall:** "Welche Aussage ist richtig?" — Akzeptabel fuer MC (impliziter Operator: "identifiziere"), aber expliziter Operator bevorzugt.

### 6.6 A9 Tafelbild-Bezug — Knoten-Mapping

**Methode:** Pro Aufgabe pruefen: Zielt diese Aufgabe auf das Verstaendnis eines bestimmten TB-Knotens? Testfrage: "Kann ein Schueler, der den TB-Knoten X nicht verstanden hat, diese Aufgabe trotzdem loesen?"

Wenn ja → kein TB-Bezug. Aufgabe prueft anderes Wissen.
Wenn nein → TB-Bezug vorhanden. Dokumentieren: `Aufgabe N → TB-Knoten kX-Y`.

**Schwellenwert:** Mindestens 1 Aufgabe pro Mappe mit direktem TB-Bezug.

### 6.7 A10 Inhaltsgesteuerte Typauswahl — Begruendungspruefung

**Methode:** Pro Mappe die Typverteilung tabellarisch erfassen und pruefen:

| Pruefschritt | Beschreibung |
|---|---|
| 1. Zaehlung | Jeden Aufgabentyp zaehlen: MC=?, ZU=?, LT=?, RF=?, FT=? |
| 2. Vielfalt | Mindestens 3 verschiedene Typen? |
| 3. Maximum | Kein Typ > 3x? |
| 4. Freitext | Freitext mindestens 1x, letzte Position, AFB III? |
| 5. Wiederholungs-Begruendung | Jeder Typ der > 1x vorkommt: Gibt es im Progressionsplan eine didaktische Begruendung? |
| 6. Operationalisierungsziel-Passung | Pro Aufgabe: Ist der gewaehlte Typ der diagnostisch praeziseste fuer das Operationalisierungsziel? |

**PASS-Muster:** 2x MC (Pos 1: Fachbegriff-Erkennung AFB I, Pos 4: Transfer-Unterscheidung AFB II), 1x LT, 1x ZU, 1x RF, 1x FT — Begruendung im Progressionsplan: "2x MC weil unterschiedliche kognitive Anforderung".
**FAIL-Muster:** 3x MC ohne Begruendung, oder LT gewaehlt wo ZU diagnostisch praeziser waere.

### 6.8 A16 Fragebogen-Kohaerenz — Sequenz-Mapping

**Methode:** Die SCPL-Zonen des Hefteintrags als Referenzsequenz nehmen und die Aufgabensequenz dagegen mappen:

1. SCPL-Zonen des Hefteintrags auflisten: S, C1, C2, ..., Cn, P, L
2. Pro Aufgabe die getestete Zone identifizieren (aus Konstruktionskontext oder TB-Knoten-Zuordnung)
3. Pruefen: Korrespondiert die Aufgabenreihenfolge mit der Zonenreihenfolge?

**PASS:** Aufgabe 1 → S, Aufgabe 2 → C1, Aufgabe 3 → C2, Aufgabe 4 → C3, Aufgabe 5 → P, Aufgabe 6 → L, Aufgabe 7 → L
**FAIL:** Aufgabe 1 → C2, Aufgabe 2 → S (Zone uebersprungen, dann nachgeholt)
**FAIL:** Aufgabe 1 → S, Aufgabe 2 → P (C-Zonen uebersprungen)

**Toleranz:** Zusammenfassung benachbarter C-Schritte in 1 Aufgabe ist erlaubt (z.B. Aufgabe 3 testet C2+C3). Ruecksprung ist nicht erlaubt.

### 6.9 A17 SCPL-Zonen-Abdeckung — Vollstaendigkeitspruefung

**Methode:** Jede SCPL-Zone als Zeile, jede Aufgabe als Spalte. Eintragen, welche Zone jede Aufgabe primaer diagnostiziert:

```
Zone    | Aufg.1 | Aufg.2 | Aufg.3 | Aufg.4 | Aufg.5 | Aufg.6 | Aufg.7 |
S       |   X    |        |        |        |        |        |        |
C1      |        |   X    |        |        |        |        |        |
C2      |        |        |   X    |        |        |        |        |
C3      |        |        |        |   X    |        |        |        |
P       |        |        |        |        |   X    |        |        |
L       |        |        |        |        |        |   X    |   X    |
```

**PASS-Bedingung:** Jede Zone hat mindestens 1 X. Keine leere Zeile.
**FAIL-Bedingung:** Zone ohne X = ungetestete Station im Lernweg.
**Aktion bei FAIL:** Fehlende Zone identifizieren, im Progressionsplan eine Aufgabe ergaenzen (sofern Aufgabenzahl < 8) oder bestehende Aufgabe um den fehlenden Inhalt erweitern.

### 6.10 A18 Material-Aktivierung — Primaerquellen-Pruefung

**Methode:** Pro Material der Mappe pruefen, ob es in mindestens 1 Aufgabe als Primaerquelle vorkommt (im Feld `material_referenz` der Aufgabe):

| Material-ID | Typ | Als Primaerquelle in Aufgabe(n) | Nur in Tipps? | Status |
|---|---|---|---|---|
| mat-3-1 | DT | Aufgabe 1, 2 | Nein | PASS |
| mat-3-2 | BQ | — | Ja (Tipp Aufgabe 1) | **FAIL** |
| mat-3-3 | BQ | — | Ja (Tipp Aufgabe 3) | **FAIL** |
| mat-3-4 | QT | Aufgabe 3, 4 | Nein | PASS |
| mat-3-5 | TB | Aufgabe 5 | Nein | PASS |

**FAIL-Regel:** Bildquellen (BQ) und Quellentexte (QT) die NUR in Tipps vorkommen = FAIL. Sie muessen eigenstaendig als Aufgabengrundlage dienen (z.B. Bildbeschreibung, Quellenanalyse, Perspektiven-Zuordnung).
**Aktion bei FAIL:** Nicht-referenziertes Material als Ziel-Material in eine verbleibende Aufgabe einbauen. Bei BQ: Zuordnungs- oder MC-Aufgabe mit Bildanalyse. Bei QT: Zuordnungs- oder Reihenfolge-Aufgabe mit Quellenvergleich.
